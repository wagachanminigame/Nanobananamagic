### 🔥 もっとガチで使いたい人向け！有料 API キーの取得方法

「無料版の制限なんて気にしたくねぇ！」

「自分のサイトやアプリに組み込みてぇ！」

「ローカルでガンガン開発したい！」

そんな君には、**有料の Gemini API キー** を使う方法があるぜ！

#### 📝 有料 API キーの取得手順

**Step 1: Google AI Studio で API キーを作成**

1. [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセス

2. Google アカウントでログイン

3. 「Create API key」をクリック

4. プロジェクトを選択（または新規作成）

5. API キーが発行されるから、コピーして保存！

**Step 2: Google Cloud Console で課金を有効化**

ここが重要なポイントだぜ！

API キーを作っただけじゃ、無料枠の制限がかかったままなんだ。

課金を有効にすることで、制限が解除されるぞ！

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセス

2. 左上のメニューから「お支払い」を選択

3. 「請求先アカウントを作成」をクリック

4. クレジットカード情報を登録

5. Step 1 で使ったプロジェクトに請求先アカウントをリンク

これで準備完了だ！

**Step 3: Nanobanana Magic で API キーを設定**

1. Nanobanana Magic を開く

2. 🔑 ボタンをクリック

3. 取得した API キーを入力

4. 「保存」をクリック

これで、制限なしでガンガン生成できるようになるぜ！

#### 💰 料金の目安

有料 API は **従量課金制** だから、使った分だけ請求される仕組みだ。

画像生成は 1 回あたり数円〜数十円くらいが目安だぜ。

詳しい料金は [Google AI の料金ページ](https://ai.google.dev/pricing) をチェックしてくれ！

#### ⚠️ 注意点

- **使用モデル**: `gemini-3-pro-image-preview`

- 無料の API キーでは画像生成ができないから注意！

- 課金を有効にしないと `limit: 0` のエラーが出るぞ！
