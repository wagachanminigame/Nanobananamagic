## 🏭 グッズ化の手順（pixivFACTORY）

Nanobanana Magic で作った画像を、**実際のグッズとして販売**できます！

### Step 1: 画像を生成

1. グッズに合ったテーマを選ぶ
   - 例: 「【グッズ】T シャツデザイン」を選択
2. 呪文を入力して生成
3. 画像をダウンロード

### Step 2: pixivFACTORY でグッズ化

1. [pixivFACTORY](https://factory.pixiv.net/) にアクセス
2. 作りたいグッズを選ぶ
3. 生成した画像をアップロード
4. 位置・サイズを調整
5. 注文 or BOOTH で販売！

### 対応グッズと推奨設定

| グッズ | テーマ選択 | 自動設定される比率 |
|--------|-----------|-------------------|
| T シャツ | 【グッズ】T シャツデザイン | 縦長 (3:4) |
| アクリルスタンド | 【グッズ】アクリルスタンド | 縦長 (3:4) |
| スマホケース | 【グッズ】スマホケース | スマホ全画面 (9:16) |
| マグカップ | 【グッズ】マグカップ | ワイド (16:9) |
| トートバッグ | 【グッズ】トートバッグ | 正方形 (1:1) |
| キーホルダー | 【グッズ】アクリルキーホルダー | 正方形 (1:1) |
| 抱き枕カバー | 【グッズ】抱き枕カバー | スマホ全画面 (9:16) ※表裏 2 枚自動生成 |

## 📝 基本操作

### クイックスタート

```
1. 呪文を入力 → 例:「宇宙を漂う金魚」
2. 世界観を選択 → 例:「水彩画風」
3. CREATE ITEM をクリック
4. 10〜30 秒待つ
5. 完成！ダウンロード or シェア
```

### AUTO CRAFT（大量生成）

```
1. 設定を決める
2. 「AUTO CRAFT」ボタンをクリック
3. 最大 20 枚まで自動生成
4. 気に入ったものを選ぶ
```

## ⚠️ 注意事項

### API について

- **画像生成には有料 API キーが必須**です（ローカル実行の場合）
- 無料 API キーでは `limit: 0` のエラーが出ます
- **無料で使いたい場合は [Google AI Studio](https://ai.studio/apps/drive/1fFjmvZbE4HWDvqrfhbLUD8vvICJymf58) をご利用ください**

### 生成について

- 生成には 10〜30 秒かかります
- 内容によっては AI が生成を拒否する場合があります
- 抱き枕カバーは自動で表裏 2 枚生成されます

### グッズ販売について

- AI 生成画像の商用利用は各自の責任で行ってください
- pixivFACTORY の利用規約を確認してください
- 著作権のある既存キャラクターの生成は避けてください
